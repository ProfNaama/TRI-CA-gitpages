Download TRI-CA

To download and run TRI-CA locally, follow these steps:

  1. Visit the official GitHub repository: TRI-CA on GitHub
  2. Copy the repository URL.
  3. Clone the repository to your local machine using Git: 
    git clone https://github.com/ProfNaama/TRI-CA.git
  4. Navigate into the project directory, install dependencies with npm install, and follow the Quick Start instructions in the repository README.
